# Seagrass Project

This repository contains data and R scripts for the analysis of seagrass acroos Caribbean.

## Folder Structure
- `RawData/`: Contains all raw CSV data files.
- `IntermediateData/`: Contains processed data files.
- `RCode/`: Contains R scripts for data analysis.

## Usage
1. Clone the repository.
2. Use R scripts in `RCode/` to process and analyze data.
3. Large files are tracked with Git LFS.
